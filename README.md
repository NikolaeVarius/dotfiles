# My dotfiles
Requires GNU Stow: https://www.gnu.org/software/stow/

Useful Reference: https://alexpearce.me/2016/02/managing-dotfiles-with-stow/

## Using
1. Clone into your home directory
2. `mv dotfiles/ .dotfiles/ && cd .dotfiles/`
3. `stow .vim` etc etc . . .
